## 2. 设计（缝去趋势）

谱修正的输入矩场 φ(y,x) 在边界行处有 wrap 不连续（缝）。对每列 x：

```text
J(x) = [φ(0)−φ(ny−1)] − ½[(φ(1)−φ(0)) + (φ(ny−1)−φ(ny−2))]   # 缝的「超额跳变」
trend(y,x) = J(x)·s̃(y)，s̃ = 零均值单位超额跳变锯齿
校正输出 = apply_correction(φ − trend) + trend                   # 趋势项直通、不被修正
```

性质：(i) 光滑周期场 `J=O(Δ³∂³φ)≈0` → 与原修正差 ~roundoff（标定 k=0.098 行为不变，且该 k 本就在 smoothstep 带下缘之外）；(ii) 有缝场的 1/k 谱泄漏尾降为 ≥1/k²，切断「修正 × 缝」注入通道；(iii) s̃ 零均值 → 全局矩守恒不变。

## 6. b′ 范围升级：窗化全局算子（2026-07-04 用户批准，设计换代）

**准则转变**：近壁算子行为不可能既修缝又逐位保持（施加行折点与热边界层同居）；设计准则改为「按构造保护内部/标定行为 + **经验重验近壁链**」。

**机制（一个机制同治三源）**：对每个全局周期算子（碰撞内 4 处 FFT 修正 + 双调和滤波），当边界条带配置非零时：

```text
w(y)  = 边界条带平滑窗：底部 seam_aware_bottom_rows 行与顶部 seam_aware_top_rows 行 w=0，
        经 seam_aware_taper_rows 行余弦缓升到 1，内部恒 1
ψ     = φ̄ + w·(φ − φ̄)          # 施加行内容与全部折点/wrap 跳变不进谱分析（φ̄=全域均值）
输出  = φ + w·[C(ψ) − ψ]        # 修正增量以 w 掩码写回；施加行不受写入（反正每步重构）
```

性质：内部（w=1 且远离缓升区）修正行为=冻结路径作用于 ψ≈φ；施加行增量恒 0；wrap 与内部折点均被窗零化——修正缝（~75%）与滤波缝（~17%）同机制处理。近壁缓升区（含热边界层前几行）的修正强度改变——**这就是被接受的 M3 风险**，由 G4′ 经验重验兜底。

**配置**：`collision.seam_aware_bottom_rows / seam_aware_top_rows / seam_aware_taper_rows`（默认全 0 = off，冻结路径逐位不变；仅 Phase_4 派生配置经 `collision_overrides` 启用）。

## 9. D3 第一刀:粗网格声学区(2026-07-05,核心正面)

D1 消融证明 dispersion 是热专用的 → 声学粗网格区不需要它 → 无缝注入源。第一刀验证"介质本身干不干净"(剥离开边界实现)。探针 `scripts/phase4_d3_coarse_acoustic_probe.py`。

**核心正面**(tuned-tau 粗网格、无 dispersion、纯声学出射脉冲的回散射):

| 配置 | tau21 | w-/w+ 回散射 | 声速误差 | 振幅存活 |
|---|---|---|---|---|
| dx=334μm,λ/dx=104,ν×100 | 0.573 | **0.0018** | +0.14% | 99% |
| dx=334μm,λ/dx=104,ν×200 | 0.646 | **0.0030** | +0.36% | 99% |

对照:dx2p6 注入底板 0.15、P4-1 门 0.05、seam-free 理想 0.001。**声学区回散射 ~0.002,比注入底板低 30×、低于门 25×——体积注入底板在声学区不存在。**

**暴露的三条 D3 工程复杂度(诚实)**:
1. 声学区不能"改 dx"——朴素粗化(保冻结映射)tau→0.5007,声学变脏(0.08)/崩;必须独立配 tau(人为粘性,绕过 unit_mapping 的 Pr/tau32>tau21 绑定)→ 声学区是**独立配置的 LBM 域**。
2. 碰撞的 heat-flux 最小范数正则化大粘性下 gram 奇异 → 声学区应**关掉它**(声学不需热流精度)→ 简化碰撞。
3. 细热区↔粗声区**界面耦合**未碰(标准但复杂的 grid-refinement,第二刀)。

**判决**:D3 核心断言成立(声学介质干净),剩余=多域工程(独立声学域 + 界面耦合),**tractable engineering,非 rabbit hole**(区别于 D1/b′ 的根本性死)。

**D3 完整路线图**:① 独立声学域配置(简化碰撞、tuned tau、粗网格)—已验证介质干净 → ② 声学区开边界(无注入源,现有 open_cbc 或 sponge 应能过 |R|<0.05)→ ③ 界面耦合(细 M3 源↔粗声区,最难)→ ④ 端到端 M3 源→界面→声学区→开边界→远场。
