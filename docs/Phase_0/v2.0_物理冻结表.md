## 1. 冻结范围总表

| 类别 | 冻结决策 | 实现含义 | 不做/边界 | 状态 |
|---|---|---|---|---|
| 研究对象 | 悬空 CNT 薄膜热声换能器 | 薄膜作为零厚度表面热源，气体在薄膜两侧受热产生热声响应 | 不做基底支撑、不做封装结构 | 已冻结 |
| 几何维度 | 2D freestanding 矩形条带 | x 为薄膜方向，y 为法向；优先使用关于薄膜中面的半域对称 | 第一篇不做 3D 和阵列 | 已冻结 |
| 薄膜力学 | 刚性薄膜、无机械振动 | 声辐射只来自热致气体膨胀，不求解薄膜结构动力学 | 不做热弹耦合、预应力振动 | 已冻结 |
| 薄膜热模型 | 集总面热容模型 | `C_A dT_s/dt = P_in(t) - 2 q''_g(t)` | 不解薄膜内部导热方程 | 已冻结 |
| 气体模型 | 完全可压缩 Navier-Stokes-Fourier 极限 | LBM 需恢复质量、动量、能量方程和理想气体状态方程 | 不做稀薄气体修正 | 已冻结 |
| 气体种类 | 空气为主，理想多原子气体 | 目标 `gamma = 1.4`，`Pr = 0.706` | 其他气体只作后续扩展 | 已冻结 |
| 驱动方式 | 直接给定面热功率 `P_in(t)` | 跳过电学层，默认正弦、阶跃、高斯脉冲三类输入 | 不做电流、电阻、Joule 热细节 | 已冻结 |
| 默认偏置 | `P_bar = 0` | 主线先做零均值正弦热功率扰动 | 有直流偏置的实验复现实例后置 | 已冻结 |
| 界面速度 | 无滑移/刚壁 | `u_Gamma = 0` | 不做热蠕移和稀薄边界 | 已冻结 |
| 界面热条件 | Level A/B/C 三级渐进 | A 给定温度，B 给定热流，C 薄膜 ODE 与 LBM 共轭 | Phase 1-2 不直接上 Level C | 已冻结 |
| 辐射/远场 | 近场 LBM + 远场 Kirchhoff 外推 | LBM 只覆盖热黏性近场，远场由控制面数据外推 | 若 Phase 4 受阻，论文可收缩为近场结果 | 已冻结 |
| 开边界 | CBC 为主，sponge layer 备用 | 目标反射系数 `R_abs < 0.05` | 不把大域直接算到远场 | 已冻结 |
| 速度集 | D2Q21 起步 | Phase 2 先实现 D2Q21；速度向量、权重、温标单独冻结 | D2Q17 不再作为主线 | 已冻结 |
| 高精度通道 | D2Q37 或等价九阶速度集 | 若 D2Q21 的 `Pr/gamma` 验证不能达标，则升级 | 不在 Phase 0 直接实现 D2Q37 | 已冻结 |
| 碰撞模型 | SMRT + binomial transform | 独立控制二阶/三阶松弛，恢复可调黏性、热扩散与 Pr | 不使用 BGK 作为主线模型 | 已冻结 |
| 多原子闭合 | f-g 双分布内部自由度方案 | `f` 携带平动能，`g` 携带额外内部自由度 | 不用单分布强行拟合 gamma | 已冻结 |
| 代码策略 | 全新构建 | NumPy + Numba，YAML 配置，HDF5 输出 | 弃用旧代码 | 已冻结 |

## 3. 驱动与验证工况

| 工况 | 表达式 | 主要用途 | 输出量 |
|---|---|---|---|
| 正弦稳态 | `P_in(t) = P_bar + P_1 cos(Omega t)` | 频响、相位、线性/弱非线性偏离 | `T_s`、`q''_g`、`p'` 幅值和相位 |
| 阶跃输入 | `P_in(t) = P_bar H(t - t_0)` | 启动瞬态与时间常数 | `T_s(t)`、`p'(t)`、能量平衡 |
| 高斯脉冲 | `P_in(t) = P_bar exp(-(t-t_0)^2/sigma^2)` | 冲激响应核与宽频检查 | 近场脉冲响应、频谱 |

## 4. 界面层级冻结表

| 层级 | 边界条件 | 薄膜方程 | 用途 | 通过标准 |
|---|---|---|---|---|
| Level A | `u=0`, `T_Gamma = T_s(t)` | 不解薄膜 ODE | 等温壁验证 | 与 Phase 1 参考解误差 < 5% |
| Level B | `u=0`, `-k_g dT/dn = q''(t)` | 不解薄膜 ODE | 给定热流验证 | 与 Phase 1 参考解误差 < 5% |
| Level C | `u=0`, `T_Gamma = T_s(t)` | `C_A dT_s/dt = P_in - 2q''_g` | 主线共轭耦合 | 近场声压、相位、能量残差达标 |

## 5. 决策门槛

| 门槛 | 检查内容 | 数值标准 | 决策 |
|---|---|---|---|
| M1 | 1D 连续介质参考模型 vs 解析模型 | 线性区误差 < 5% | 进入 LBM 核心 |
| M2 | 热 LBM 五项验证 | 全部通过 | 进入界面耦合 |
| M2-Critical | D2Q21 的 `Pr/gamma` 验证 | 误差 < 5% | 通过则保留 D2Q21，否则修正或升 D2Q37 |
| M3 | Level A/B 界面验证 | 幅值误差 < 5%，相位误差 < 5 deg | 进入 Level C |
| M4 | CBC 与 Kirchhoff 外推 | `R_abs < 0.05`，远场误差 < 10% | 进入物理结果生产 |
| M5 | 参数扫描 | 至少 3 组与参考趋势一致 | 进入论文撰写 |
