## 1. 我对本项目内容的归纳

本项目的核心不是“普通 CNT 热声换能器仿真”，而是建立一个**可验证的时域热声 LBM 框架**。第一篇论文的主线是：

> 2D freestanding CNT 薄膜作为零厚度表面热源，气体侧由完全可压缩 NSF 极限描述；薄膜用集总面热容 ODE；近场以 D2Q21 + SMRT + f-g 双分布 LBM 起步，必要时进入 D2Q37 或等价九阶速度集 fallback，远场声压由 Kirchhoff 控制面积分外推。

**2026-06-06 状态更新**：Phase_2 M2-Critical 已实际触发 D2Q37 fallback，并已推进到 `D2Q37_DIAGNOSTIC_READY`。该更新不改变 Phase_0 物理冻结值，也不返工 Phase_1 参考模型、CSV、manifest 或封版报告。

物理链条是：

```text
P_in(t)
  → 薄膜温度 T_s(t)
  → 壁面热流 q''_g(t)
  → 热边界层内气体膨胀/压缩
  → 近场 p'(x,y,t)
  → 控制面数据
  → Kirchhoff 远场声压
```

Phase_1 的任务不是写 LBM，而是建立一个**1D 连续介质参考模型**，作为后续 LBM 的真值基准。也就是说，Phase_1 是整个项目的“基准层”和“验收层”，不是可选项。

这张表对 Phase_1 已经足够。Phase_1 需要的不是 D2Q21 细节，而是：物性、薄膜 ODE、边界条件、验证工况和 M1 门槛。这些都已经有了。

**主要缺口只有一个：**  
`P_in(t)` 的物理含义需要再明确。现在默认 `P_bar = 0` 且 `P_in = P_1 cos(Ωt)`。作为**线性扰动热功率**，这是可以的；但作为**绝对 Joule 热功率**，它会出现负功率半周期，不物理。

建议在 Phase_1 中明确：

```text
线性频响模型中：
P_in'(t) = Re[P_hat exp(i Ω t)]
是相对于基态的扰动热功率，可正可负。

非线性/真实功率扫描中：
P_abs(t) = P_DC + P_1 cos(Ω t)，且 P_DC >= P_1。
```

这个问题不阻止 Phase_1，但必须在 Phase_1 第一份规格文件里写清楚。

## 4. 进入 Phase_1 前必须补的 6 个小项

这些不是 Phase_0 返工，而是 Phase_1 开工规格。建议写成 `Phase1_reference_spec.md`。

### 4.1 固定热功率口径

建议写成：

```text
在线性频响中，P_in 表示扰动热功率 P'_in。
P'_in(t) = Re[P_hat exp(i Omega t)]，可正可负。

在非线性功率扫描中，P_abs(t) = P_DC + P_1 cos(Omega t)，且 P_DC >= P_1。
默认 Phase_1 只验证线性扰动模型。
```

### 4.2 固定壁面热流符号

建议采用：

```text
上半域 y > 0，法向 n = +e_y。
q''_g = - k_g (∂T/∂y)|_{y=0+}

若薄膜温度高于气体，∂T/∂y < 0，因此 q''_g > 0。
薄膜 ODE：
C_A dT_s/dt = P_in - 2 q''_g
```

这个定义最自然，也和你现在的薄膜能量方程一致。

### 4.3 固定复幅值约定

建议采用：

```text
x(t) = Re[ x_hat exp(i Omega t) ]
```

则：

```text
peak amplitude = |x_hat|
RMS amplitude  = |x_hat| / sqrt(2)
phase          = arg(x_hat)
```

所有 Phase_1、Phase_2、Phase_5 后处理都按这个来。

### 4.4 固定 `beta_0` 处理

物理冻结表里已经指出 Lim 2013 解析模型可能含 `beta_0`。建议 Phase_1 这样处理：

```text
主线：beta_0 = 0
用途：对应 freestanding、忽略辐射/自然对流的核心模型。

附录/敏感性：beta_0 = 16 W/(m^2 K)
用途：只作为 Lim-type 模型对照，不混入主线 M1。
```

不要在主线 M1 里混用 `beta_0 = 0` 和 `beta_0 ≠ 0`，否则误差来源不清楚。

### 4.5 固定 Phase_1 模型层级

建议 Phase_1 分两层，不要一开始就写很重的 full 1D NSF：

```text
Phase_1a：频域线性解析/半解析参考
- Level A：给定 T_s
- Level B：给定 q''
- Level C：薄膜 ODE + 气体热导纳
- 输出复幅值和相位

Phase_1b：1D 有限差分时域参考
- 用于阶跃、脉冲、ODE 耦合、能量残差
- 与 Phase_1a 在正弦稳态下对齐
```

这样 M1 更稳，也更快。

### 4.6 固定 1D 参考模型的边界和输出位置

建议：

```text
y ∈ [0, L_y]
L_y >= 10 delta_T，低频算例按最低频率扩展

边界：
y = 0：Level A/B/C 壁面条件
y = L_y：T' = 0 或吸收/足够远边界，确保不影响壁面热流

输出：
T_s_hat
q_g_hat''
p_hat(y_c)
phase(T_s), phase(q_g''), phase(p)
energy residual
```

Phase_1 不需要完美模拟 2D 远场；它的职责是给后续 LBM 的近场热-声耦合提供参考基准。
