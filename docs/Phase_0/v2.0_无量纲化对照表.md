## 5. 热边界层定义冻结

10 kHz 默认工况采用以下冻结定义：

```text
alpha_0 = k_g/(rho_0 c_p) = 2.223e-5 m^2/s
delta_T = sqrt(2 alpha_0/Omega) = 26.6 um
```

该定义与黏性穿透深度 `delta_v = sqrt(2 nu_0/Omega)` 采用同一谐波扩散尺度口径。由于 `Pr < 1`，热扩散快于黏性扩散，因此 `delta_T > delta_v`，这也是后续网格和控制面位置的默认依据。

## 6. 频率扫描缩放关系

| 量 | 随频率缩放 | 用途 |
|---|---|---|
| `delta_T` | `Omega^(-1/2)` | 热边界层、网格需求 |
| `delta_v` | `Omega^(-1/2)` | 黏性边界层 |
| `Lambda = a/delta_T` | `Omega^(1/2)` | 有限尺寸效应 |
| `k delta_T` | `Omega^(1/2)` | 热层声学紧致度 |
| `k a` | `Omega` | 辐射指向性/远场外推 |
| `Pi_C` | `Omega^(1/2)` | 薄膜储热相位滞后 |
| `epsilon_P` | `Omega^(-1/2)` | 同功率幅值下温升尺度 |
| 每周期步数 | `Omega^(-1)` | 计算成本 |

## 7. 实现检查项

| 检查项 | 要求 |
|---|---|
| 物性表 | `rho_0, T_0, p_0, c_0, gamma, Pr, nu_0, alpha_0, k_g` 集中在配置文件中 |
| 单位转换 | 每个算例保存 SI 单位和 lattice unit 双份元数据 |
| D2Q21 温标 | `theta_0_lu = 2/3` 来自速度集冻结表，不能在碰撞模块里另行硬编码 |
| 频率扫描 | 每个频率重新计算 `delta_T, delta_v, Pi_C, epsilon_P, k a` |
| 后处理口径 | 幅值、RMS、复幅值、SPL 分开命名 |
| 能量平衡 | 周期稳态检查 `P_in - 2q''_g - C_A dT_s/dt` 的残差 |
