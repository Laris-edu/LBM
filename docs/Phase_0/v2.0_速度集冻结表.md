# CNT 薄膜热声换能器 LBM 速度集冻结表

**版本**：v2.0 Phase 0  
**日期**：2026-05-18  
**适用范围**：Phase 2 热可压缩 LBM 核心实现  
**对应模块**：`lattice_d2q21.py`

---

## 1. 冻结决策

本项目第一版 LBM 核心采用 **D2Q21 multispeed on-lattice 速度集**。D2Q17 不再作为主线实现；若 D2Q21 在 Phase 2 的 `Pr/gamma` 验证或四阶 Hermite 相关矩检查中失败，则按主计划进入 D2Q37 或等价九阶速度集应急通道。

**冻结口径**：
- 速度坐标使用格子速度单位，`c = Delta x / Delta t`
- 参考温度/声速平方：`theta_0_lu = c_s^2 = 2/3`
- 速度顺序固定为：静止、轴向1、对角1、轴向2、对角2、轴向3
- 所有权重为每个离散速度的权重，而不是每组总权重

## 4. 必须通过的矩条件检查

实现时必须提供自动测试函数，例如 `assert_d2q21_moments(tol=1e-12)`。以下检查全部通过后，速度集才允许进入平衡态和碰撞模块。

| 检查 | 目标值 |
|---|---:|
| `sum_a w_a` | `1` |
| 所有奇数总阶矩，最高到 7 阶 | `0` |
| `sum_a w_a cx_a^2`，`sum_a w_a cy_a^2` | `theta_0 = 2/3` |
| `sum_a w_a cx_a cy_a` | `0` |
| `sum_a w_a cx_a^4`，`sum_a w_a cy_a^4` | `3 theta_0^2 = 4/3` |
| `sum_a w_a cx_a^2 cy_a^2` | `theta_0^2 = 4/9` |
| `sum_a w_a cx_a^6`，`sum_a w_a cy_a^6` | `15 theta_0^3 = 40/9` |
| `sum_a w_a cx_a^4 cy_a^2`，`sum_a w_a cx_a^2 cy_a^4` | `3 theta_0^3 = 8/9` |

**验收容差**：双精度下绝对误差 `<= 1e-12`。

## 5. 与物理单位的连接

| 量 | 当前冻结值 | 说明 |
|---|---:|---|
| `Delta x` | `4 um`（10 kHz 默认工况） | 参数扫描中可按频段调整 |
| `Delta t` | `3 ns`（10 kHz 默认工况） | CFL 约 0.26 |
| 物理格子速度 | `Delta x/Delta t = 1333 m/s` | D2Q21 整数速度乘该尺度 |
| `c_0_lu` | `0.260` | `c_0 Delta t / Delta x` |
| `theta_0_lu` | `2/3` | 速度集参考温度 |

## 7. Phase 2 实现备注

- `lattice_d2q21.py` 应同时输出整数速度、浮点权重、`theta0`、反向速度索引 `opposite`
- 所有权重以精确分数或高精度浮点常量写入，避免手输小数截断
- 平衡态模块不得重新定义 `theta_0_lu`，只能从速度集模块读取
- 若矩检查失败，禁止继续运行验证算例
