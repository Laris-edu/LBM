- 多尺度网格明确采用 **方案 E（近场 LBM + Kirchhoff 远场外推）**
- 代码策略明确为**全新构建，弃用旧代码**

## 1. 项目目标

### 1.1 总目标

建立一个**经过严格验证、可发表、可持续扩展**的 CNT 薄膜热声换能器**时域数值框架**，刻画：

1. 电热输入 → 薄膜温度响应 T_s(t)
2. 固-流界面热交换 → 邻近气体热黏性响应
3. 近场热-声耦合 → 控制面声学量
4. 远场声压 p′(r→∞) → 通过 Kirchhoff 积分外推获得
5. 频率、功率、薄膜热容等参数对效率、相位、瞬态响应的影响

### 1.2 第一篇论文范围

- **几何**：2D freestanding CNT 薄膜（矩形条带），对称双侧空气
- **物理**：完全可压缩 NSF + 表面热驱动 + 集总薄膜能量方程 + 刚性薄膜
- **驱动**：直接给定热功率 P_in(t) = P̄ + P₁·cos(Ωt)，跳过电学层
- **数值**：D2Q21 + 四阶 Hermite 平衡态 + SMRT + f-g 双分布
- **网格策略**：近场 LBM + Kirchhoff 远场外推
- **创新点**：方法学（multispeed thermal LBM + 共轭热声耦合 + 控制面外推）+ 物理结果（启动瞬态、参数景观图、中等非线性偏离）
- **不做**：3D、阵列、封装、强非线性大信号、稀薄气体修正

## 2. 物理-数学框架

### 2.1 气体域控制方程

完全可压缩 NSF 方程：

```
∂ρ/∂t + ∇·(ρu) = 0
∂(ρu)/∂t + ∇·(ρu⊗u + pI) = ∇·τ
∂(ρE)/∂t + ∇·[(ρE + p)u] = ∇·(k∇T) + ∇·(τ·u)
p = ρRT,    E = e + |u|²/2
τ = μ[∇u + (∇u)ᵀ − (2/3)(∇·u)I] + μ_b(∇·u)I
```

### 2.2 薄膜能量方程

freestanding 对称构型下的最终冻结形式：

```
C_A · dT_s/dt = P_in(t) − 2 · q″_g(t)
```

其中：
- C_A = ρ_f · c_f · h_f ≈ 7×10⁻⁴ J/(m²·K)（CNT 薄膜典型值）
- q″_g(t) 由 LBM 提取（双侧对称取一侧）
- 基底导热 q″_sub = 0（freestanding）
- 辐射 q″_rad ≈ 0（小温升假设，启动后验证 ε_T < 0.05）

### 2.3 界面条件三级渐进

**Level A**（验证用）：u|_Γ = 0，T|_Γ = T_s(t) 直接给定
**Level B**（验证用）：u|_Γ = 0，−k_g·∂T/∂n|_Γ = q″(t) 直接给定
**Level C**（主线）：u|_Γ = 0，q″_g 从 LBM 提取，T_s 由 ODE 求解，T|_Γ = T_s(t)

### 2.4 驱动信号

```
P_in(t) = P̄ + P₁ · cos(Ω · t)         （稳态正弦）
P_in(t) = P̄ · H(t − t₀)                （阶跃，用于瞬态分析）
P_in(t) = P̄ · exp(−(t−t₀)²/σ²)        （高斯脉冲，用于冲激响应）
```

默认偏置 P̄ = 0。频率记号统一使用 Ω（热功率频率）。

## 3. 数值方法

### 3.1 速度集：D2Q21 起步

**决策更新**：D2Q17 已不再作为基线速度集。Phase 2 从 D2Q21 开始实现，并在进入碰撞模型前先冻结速度向量、权重、温度标度与矩条件测试。

**速度向量与权重**：
- 采用《速度集冻结表》中的 D2Q21 multispeed 方案
- 速度向量、权重、参考温度 θ₀ = 2/3、格子速度 c = Δx/Δt 以《速度集冻结表》为准
- `lattice_d2q21.py` 必须包含自动矩条件检查，至少验证 0–4 阶平衡态相关矩，以及应力、热流相关矩

共 21 个离散速度。求积精度与可恢复矩阶数以冻结表和矩条件测试为准。

**应急通道**：在 Phase 2 完成 Pr 数和 γ 验证后，若任一项误差 > 5%，优先检查 D2Q21 权重、温度标度和平衡态实现；若修正后仍不能达标，则升级到 D2Q37 或等价九阶速度集。这个决策必须在第 6 周末做出。

### 3.2 平衡态分布函数（四阶 Hermite 展开）
#Hermite19 

```
f_a^(eq) = w_a · ρ · Σ_{n=0}^{4} (1/n!) · a^(n)(ρ,u,T) : H^(n)(ξ_a)
```
$$f_{a}^{}\left(eq\right)=w_{a}\cdot\rho\cdot\Sigma_{n=0}^4(1/n!)\cdot a^{}\left(n\right)(\rho,u,T):H^{}\left(n\right)(\xi_{a})$$
具体形式参 Chen 等 2024 JFM 公式 (3.8) 或 Shan, Yuan & Chen 2006。需要至四阶速度乘积项与 (T−T₀)² 项。

### 3.3 多原子气体闭合：f-g 双分布

引入辅助分布函数 g_a 携带内部能量自由度：

```
ρ = Σ_a f_a
ρu = Σ_a f_a · ξ_a
ρe = (1/2)·Σ_a f_a |ξ_a−u|² + Σ_a g_a
```

内部自由度数 S = 2/(γ−1) − D。

**2D 空气**：D = 2, γ = 1.4 → S = 3，正确给出 c_v = (5/2)R。

g_a 的演化方程与 f_a 类似，但平衡态与碰撞算符不同。参 Nie, Shan & Chen 2008 PRE 实施细节。

### 3.4 SMRT 碰撞模型

```
Ω_a = −Σ_n (1/τ_n) · w_a · a_neq^(n) : H^(n)(ξ_a)
```

**输运系数映射**：
```
ν   = θ₀ · (τ_{2,1} − 1/2)         （剪切黏性）
ν_b = (2S/[D(D+S)]) · θ₀ · (τ_{2,2} − 1/2)   （体积黏性）
α   = θ₀ · (τ_{3,2} − 1/2)         （热扩散）
Pr  = ν/α = (τ_{2,1} − 1/2)/(τ_{3,2} − 1/2)
```

τ_2 与 τ_3 必须独立赋值。空气 Pr = 0.71 → τ_3 < τ_2。

实施中使用 binomial transform 在绝对参考系下求中心矩，避免插值耗散（Shan 2019, Shan, Li & Shi 2021）。

### 3.5 多尺度网格策略（方案 E）

**核心思想**：LBM 只覆盖近场，远场用 Kirchhoff 积分外推。

**LBM 域几何**（10 kHz 默认工况）：
- 横向（薄膜方向）：12 mm，含薄膜半宽 5 mm + 两侧各 3.5 mm 边缘外延
- 法向（半域，利用对称）：0.27 mm ≈ 10.2·δ_T，按 68 个格点实现
- 均匀网格 Δx = 4 μm（约 δ_T/6.65）
- 网格数：3000 × 68 = 2.04×10⁵
- 时间步 Δt ≈ 3 ns（CFL < 0.3）
- 一个 10 kHz 周期约 33,000 步，稳态需 10–20 周期

**控制面位置**：
- 距薄膜约 8·δ_T ≈ 0.213 mm 的水平面
- 在该面上记录 p′(x,t)、u_n(x,t)、∂p′/∂n(x,t) 的时间序列

**远场外推**（2D Kirchhoff 频域积分）：
```
p̂(r,Ω) = (1/4)·∮_S [p̂·∂H₀⁽¹⁾(kr')/∂n − H₀⁽¹⁾(kr')·∂p̂/∂n] dS
```
其中 H₀⁽¹⁾ 为第一类 Hankel 函数，k = Ω/c₀。

**Phase 2-3 简化版**：使用方案 A（均匀小域 + CBC 边界），不做远场外推，先把近场代码调通。Phase 4 升级到方案 E。

### 3.6 开放边界条件

LBM 域外边界使用基于 LODI 分析的特征边界条件（CBC），参 Chen, Yang & Shan 2023 JCP。目标反射系数 |R| < 0.05。

备用方案：sponge layer（耗散层）作为 CBC 调试期的应急。

### Phase 1｜参考连续介质模型（第 2–3 周）

**目标**：建立 1D 法向参考解，作为后续 LBM 真值基准。

**任务**：
1. 实现 1D NSF 有限差分求解器（不用 LBM）
2. 三个验证算例：
   - **A**：给定 T_s(t) = T̄ + T₁cos(Ωt)
   - **B**：给定 q″(t) = q̄ + q₁cos(Ωt)
   - **C**：薄膜 ODE 共轭 P_in(t) = P̄ + P₁cos(Ωt)
3. 与解析模型对照：
   - Arnold-Crandall 1917
   - McDonald-Wetsel 1978
   - Lim, Tong & Li 2013

**输出**：
- T_s、q″_g、近场 p′ 的幅值与相位 vs Ω
- δ_T、δ_v 与频率实测关系
- 不同 C_A 下的频响曲线
- 不同输入功率下的偏离曲线

**门槛 M1**：参考模型与至少一个解析模型在线性区对齐 < 5%

### Phase 2｜气体侧热 LBM 核心（第 4–7 周）

**目标**：在不接薄膜的前提下做对"热气体求解器"。

**任务**：
1. 实现 D2Q21 速度集与权重
2. 实现四阶 Hermite 平衡态（f 和 g 双分布）
3. 实现 SMRT 碰撞（τ₂、τ₃ 独立）
4. 实现 binomial transform
5. 验证测试套件

**5 项必过验证**：

| 测试 | 验证量 | 标准 |
|---|---|---|
| 1D 热扩散 | T(x,t) 与解析 Green 函数 | L₂ 误差 < 1% |
| 平面声波 | 声速 c = √(γRT)、衰减 | 误差 < 2% |
| Couette 流 | 速度剖面、ν | 误差 < 1% |
| Pr 数扫描 | 实测 Pr vs 理论 | 偏差 < 3% |
| γ 验证 | 等熵关系 p·V^γ = const | 偏差 < 2% |

**门槛 M2**：5 项全过。**若 γ 或 Pr 偏差 > 5%，第 6 周末必须决定是否修正 D2Q21 方案或升级至 D2Q37。**

### Phase 3｜界面耦合（第 8–10 周）

**目标**：把薄膜 ODE 与气体 LBM 真正接起来。

**任务**：
1. 实现 Level A 等温壁边界（multispeed 适用的 Inamuro 或 regularized BC）
2. 实现 Level B 热流壁边界
3. 实现壁面热流提取：
```
q″_w = (1/2)·Σ_a [(ξ_a−u)·(ξ_a−u) − (D+S)θ₀]·(ξ_a·n)·f_a + Σ_a (ξ_a·n)·g_a
```
4. 实现薄膜 ODE（RK4）+ 与 LBM 同步
5. 实现 Level C 共轭耦合

**验证**：
- (a) 阶跃 P_in、无气体散热 → 指数解
- (b) 正弦 P_in → 稳态振幅、相位 vs Phase 1 参考
- (c) 全耦合 → 近场 p′(x,t) vs Phase 1 参考

**门槛 M3**：Level A & B 与参考模型对齐 < 5%

### Phase 4｜开边界与远场外推（第 11–12 周）

**目标**：方案 E 落地。

**任务**：
1. 实现 CBC 开边界
2. 反射系数测试：|R| < 0.05
3. 实现控制面数据采集模块
4. 实现 Kirchhoff 频域外推
5. 端到端验证：与解析 thermophone 远场公式对照

**门槛 M4**：
- |R| < 0.05
- Kirchhoff 远场与解析远场偏差 < 10%（更宽松，因为加入了外推环节）

### Phase 5｜物理结果生产（第 13–15 周）

**目标**：4 组可发表参数扫描。

**任务**：
1. **频率扫描** Ω：覆盖 1–100 kHz，看 δ_T/δ_v、幅相、远场 SPL、效率
2. **功率扫描** P₁：从线性区扫到 ε_T ≈ 0.1，识别非线性偏离起点
3. **面热容扫描** C_A：看 Π_C 对截止频率与效率的影响
4. **启动瞬态**：阶跃 P_in，提取 T_s 与 p′ 的瞬态响应

**可选**：有限尺寸扫描 Λ = a/δ_T，看平面波到圆盘辐射过渡。

**门槛 M5**：≥ 3 组与参考模型趋势一致

## 7. 风险与备选

| 风险 | 概率 | 影响 | 备选方案 |
|---|---|---|---|
| D2Q21 精度或稳定性不足 | 中 | 高 | M2-Critical 决策点升级至 D2Q37 或等价九阶速度集 |
| 共轭界面数值不稳 | 中 | 中 | 推迟 Level C，先用 Level B 走通管线 |
| Kirchhoff 外推误差大 | 中 | 中 | 改用 FW-H 积分或加大控制面距离 |
| LBM 主循环性能不足 | 中 | 中 | Numba → CuPy GPU |
| 启动瞬态分析复杂 | 低 | 中 | 简化为冲激响应核拟合 |
| 18 周走不完 | 高 | — | 降级至"1D + Level B + 频响"作为第二论文备选 |

**降级路径**：

如 Phase 3 共轭耦合不稳，论文范围收缩为：
- "2D freestanding LBM with prescribed-flux boundary"
- 主结果：方法学 + Level B 频响验证 + 启动瞬态
- Level C 共轭耦合留作下一篇

如 Phase 4 远场外推困难，论文范围收缩为：
- "近场 LBM 热声响应的方法学"
- 不报远场 SPL，只报近场流场和薄膜响应

## 11. 要点速览（Top 12）

1. **几何**：2D freestanding，薄膜双侧对称空气，主线不做 3D。

2. **驱动**：直接给热功率 P_in(t)，跳过电学层，全部用频率 Ω。

3. **方程**：完全可压缩 NSF + 薄膜集总能量方程 C_A·dT_s/dt = P_in − 2q″_g。

4. **速度集**：D2Q21 起步，第 6 周末 M2-Critical 决策点。若 γ 或 Pr 偏差 > 5%，先修正 D2Q21 权重/温标/平衡态；仍不达标则升 D2Q37。

5. **平衡态**：四阶 Hermite，包含 (T−T₀)² 项与四阶速度乘积。

6. **多原子**：f-g 双分布，S = 3，恢复空气 γ = 1.4。

7. **碰撞**：SMRT，τ₂ 与 τ₃ 独立，binomial transform 在绝对系求矩。

8. **网格策略**：方案 E。LBM 只覆盖近场（10·δ_T 法向 + 边缘外延），Kirchhoff 外推远场。Phase 2-3 用方案 A 简化版，Phase 4 升级。

9. **界面**：三级渐进 A → B → C，主线用 C。

10. **开边界**：CBC，目标 |R| < 0.05。

11. **代码**：弃用旧代码，全新构建。Numba 加速，YAML 配置，HDF5 存储。

12. **论文**：第一篇做"方法学 + 共轭热声 + 启动瞬态 + 中等非线性偏离"。3D、阵列、封装、强非线性、电学耦合留给后续工作。
