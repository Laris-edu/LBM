# CNT 薄膜热声换能器 LBM 参数扫描计划表

**版本**：v2.0 Phase 0  
**日期**：2026-05-18  
**用途**：定义第一篇论文所需图谱、算例矩阵、计算预算与降级策略，并反向约束 Phase 1-4 的实现和验证输出。

---

## 1. 设计原则

参数扫描计划表不是参数清单，而是论文图谱和计算预算的约束文件。所有扫描必须回答三个问题：

1. 这组算例支撑哪张论文图？
2. Phase 1 参考模型是否能给出对应真值或趋势？
3. Phase 5 三周内是否有足够算力完成？

当前冻结物性沿用 Phase 0 文档：
- `delta_T(10 kHz) = 26.6 um`
- `alpha_0 = 2.223e-5 m^2/s`
- `Pr = 0.706`
- 默认 `P_1 = 1000 W/m^2`
- 默认 `C_A = 7e-4 J/(m^2 K)`
- 默认薄膜半宽 `a = 5 mm`

## 4. 分频段自适应网格策略

1-100 kHz 频率扫描不采用固定 `Delta x=4 um`。扫描范围切成四段，每段按最高频率确定 `Delta x`，保证 `delta_T/Delta x >= 5`；按最低频率确定法向域，保证覆盖约 `10 delta_T_max`。控制面统一设在域内 `y_c = 0.8 L_y`，并在结果元数据中记录实际 `y_c/delta_T(f)`。

| 频段 | 频率范围 | `Delta x` | `L_y` | 网格 `Nx x Ny` | `delta_T/Delta x` 最小值 | 控制面 `y_c` | 估算内存 |
|---|---:|---:|---:|---:|---:|---:|---:|
| A | 1-3 kHz | 8 um | 0.848 mm | 1500 x 106 | 6.07 | 0.678 mm | ~105 MB |
| B | 3-10 kHz | 4 um | 0.488 mm | 3000 x 122 | 6.65 | 0.390 mm | ~242 MB |
| C | 10-30 kHz | 2 um | 0.268 mm | 6000 x 134 | 7.68 | 0.214 mm | ~532 MB |
| D | 30-100 kHz | 1.5 um | 0.1545 mm | 8000 x 103 | 5.61 | 0.124 mm | ~545 MB |

**跨段一致性验证**：
- `10 kHz` 必须分别用 B 段与 C 段网格各跑一次
- `30 kHz` 必须分别用 C 段与 D 段网格各跑一次
- 双网格对照的 `|p_hat|`、相位、`q''_g` 差异应低于 5%；若不满足，优先加密较粗频段

## 6. 分级策略

| 等级 | 内容 | 算例数 | 是否必须 | 删除条件 |
|---|---|---:|---|---|
| Baseline minimum | Fig.1/Fig.2 + Fig.3 + Fig.5 | 33 | 必须 | 不删除 |
| Core paper | Baseline + Fig.4 | 133 | 必须 | 仅在性能严重不足时降采样 Fig.4 |
| Robustness | 跨段双网格对照 + 关键验证补充 | 20 | 必须 | 不删除，但可减少重复点 |
| Nice-to-have | `Lambda` 有限尺寸扫描 | 5 | 可选 | 只有核心图完成后执行 |

**降级方案**：
- 若总计算时间超过 Phase 5 预算，Fig.4 从 `5 x 20` 降为 `3 x 12`
- 若 Level C 共轭耦合不稳，Fig.3/Fig.4 先用 Level B 给定热流边界生产方法学结果
- 若 Kirchhoff 外推未通过 M4，只保留近场 `p'`、`q''_g`、`T_s` 的参数图，不报告远场 SPL

## 7. 对 Phase 1-4 的反向要求

| 阶段 | 反向要求 |
|---|---|
| Phase 1 | 1D 参考模型必须支持 Fig.1、Fig.3、Fig.4 的同一参数网格，并输出幅值、相位、热流、能量残差 |
| Phase 2 | 配置系统必须能批量生成频段网格、物性、`P_1`、`C_A` 的 YAML |
| Phase 3 | Level A/B/C 都必须输出统一后处理字段，方便复用同一脚本画图 |
| Phase 4 | 控制面数据必须记录 `y_c`、`y_c/delta_T`、频率、网格段、Kirchhoff 输入量 |
| Phase 5 | 每个算例保存元数据、参考模型对照结果和失败原因，便于预算复核 |
